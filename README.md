# PHP-reservation-app
School assignment in PHP

### Install

1. **requirements:**
    - Nodejs
    - git
2. **clone:**
   ```
   git clone https://github.com/azerupi/PHP-reservation-app.git
   cd PHP-reservation-app
   ```
3. **Install dependencies**
   ```
   npm install
   ```
4. Run a webserver, navigate to the project, enjoy!
